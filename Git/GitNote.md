


##### Git tag
```
    // Show list tag
    git tag -l "v1.x.x"

    // Show tag detail
    git show v1.x.x

    // Create tag
    git tag -a v1.x.x -m "message"

    // Push tag
    git push origin v1.x.x
    
    // Delete tag
    git tag -d v1.x.x // local
    git push origin --delete v1.x.x // remote
```
